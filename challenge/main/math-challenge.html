About this project

This project is created so that you have the possibility to look at some math stuff outside of school
The design of the task is chosen in a way that you have to understand the task and then have to use some python
scripts to find the solution. Once you found a solution change the url and another problem will occur
In the next level you will always have the possibility to look at our solution of the problem beforehand.
We will provide further explaination for each problem to make sure you really unterstood it.
You can acess this solution page by adding _solution to the the current website (e.g. math-challenge_solution.html)
Make sure to sometimes check the html code of the website as it could contain some hints :).

We will ask for the shortest solution solution possible.
